# Техническое задание: Учет студентов группы

## 1. Цель работы
Создать программу для хранения информации о студентах группы.

## 2. Что делает программа
- Добавляет нового студента
- Показывает всех студентов
- Ищет студента по фамилии
- Удаляет студента

## 3. ER-диаграмма (схема данных)

┌─────────────────┐
│    Student      │
├─────────────────┤
│ ID (главный ключ)│
│ Фамилия         │
│ Имя             │
│ Возраст         │
│ Средний балл    │
└─────────────────┘

## 4. Что храним в таблице

| Поле | Тип данных | Описание |
|------|------------|----------|
| id | int | Номер студента |
| last_name | string | Фамилия |
| first_name | string | Имя |
| age | int | Возраст |
| gpa | float | Средний балл |

## 5. Функции программы
1. Добавить студента
2. Показать всех
3. Найти по фамилии
4. Удалить студента
5. Выход
